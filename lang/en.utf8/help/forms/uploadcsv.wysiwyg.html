HTML editor

An HTML editor is available in some sections of the site. This is known as a 'What you see is what you get' (WYSIWYG) editor. This allows the users to apply formatting to their texts, e.g. bold or italics. If the editor is turned off, the users can only enter plain text.

This setting might be disabled by a site administrator.

Note: HTML editors may appear and work slightly differently in some browsers.
