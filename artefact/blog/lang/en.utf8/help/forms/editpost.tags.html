Tags

You can add tags to artefacts and pages you create. Tags allow you to add descriptive labels to artefacts and pages and create an index of them.

To associate multiple tags with an artefact or page, separate each tag entry with a comma. For example – ‘Assessment, Semester 2, Management’.

Clicking on Show my tags will generate a list of the tags you have previously used within your portfolio and the number of times they have been used. Clicking on an existing tag will add it to the tag list for the artefact or page you are currently creating or editing.
